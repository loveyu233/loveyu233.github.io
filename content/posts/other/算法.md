# 一: 指针

## 1.1: 快慢指针

>慢指针一次移动一格,快指针一次移动k格
>
>典型问题:	判断一个链表是否有环
>
>但是如果就只问有环链表，显得太基础 (￣_,￣ )
>
>于是，就有了一些抽象有环链表的问题，但是这些问题大多数都有一个明显的特性，就是在移动的过程中会出现一个环，这个环可以是一组周期出现的数字，也可以就是沿线性表移动过程中出现的环
>
>所以只要出现这个明显的特性就可以想到弗洛伊德的龟和兔子

### [141. 环形链表](https://leetcode.cn/problems/linked-list-cycle/)

>最基本的快慢指针题,直接按定义就可以

```go
// 141
func hasCycle(head *ListNode) bool {
	var left *ListNode = head
	var right *ListNode = head
	for right != nil && right.Next != nil {
    // 必须先移动快慢指针再判断是否快慢指针是否一样
		left = left.Next
		right = right.Next.Next
		if left == right {
			return true
		}
	}
	return false
}
```

### [142. 环形链表 II](https://leetcode.cn/problems/linked-list-cycle-ii/)

>第一步判断是否为环形链表
>
>第二部一个指针从相遇节点出发一个指针从头节点出发,当相遇时即为环形链表的开始节点

```go
// 142
func detectCycle(head *ListNode) *ListNode {
	slow, fast := head, head
  // 第一个循环判断是否是环形链表
	for fast != nil && fast.Next != nil {
		slow = slow.Next
		fast = fast.Next.Next
    // 如果是环形链表
		if slow == fast {
      // fast从头开始,slow从相遇位置开始,两个都每次前进一个,当再次相遇时该节点就是环形节点的入口
			fast = head
			for slow != fast {
				slow = slow.Next
				fast = fast.Next
			}
			return fast
		}
	}
	return nil
}
```

### [202. 快乐数](https://leetcode.cn/problems/happy-number/)

>隐性快慢指针题,该题的要求是最后结果为1即为快乐数也就是判断该链表是否为环形,如果是环形呢必然不是快乐数,只有不是环形且快指针结果为1才是快乐数

![kuaileshu](../../../img/kuaileshu.png)

```go
func isHappy(n int) bool {
	slow, fast := n, step(n)
  // 不要slow != fast这个条件也是可以的,因为当fast为1时无论后边多少次前进依旧还是1,slow始终会追赶上也等于1所以不要也行
	for fast != 1 && slow != fast {
		slow = step(slow)
		fast = step(step(fast))
	}
	return fast == 1
}
func step(n int) int {
	nums := 0
	for n > 0 {
		nums += (n % 10) * (n % 10)
		n = n / 10
	}
	return nums
}
```

### [287. 寻找重复数](https://leetcode.cn/problems/find-the-duplicate-number/)

>该题和环形链表II一样的思路,因为题目中规定了该数组一定有相同数和该数组的任意值不超过数组的索引值所以可以用数组值当做索引值

```go
func findDuplicate(nums []int) int {
	slow, fast := 0, 0
	for {
		slow = nums[slow]
		fast = nums[nums[fast]]
		if slow == fast {
			fast = 0
			for {
				if slow == fast {
					return slow
				}
				slow = nums[slow]
				fast = nums[fast]
			}
		}
	}
}
```

### [876. 链表的中间结点](https://leetcode.cn/problems/middle-of-the-linked-list/)

>fast每次前进一步slow每次前进一步,当fast等于空或者下一个节点为空时slow就是该链表的中间节点

![middhandler](../../../img/middhandler.png)

```go
func middleNode(head *ListNode) *ListNode {
	slow, fast := head, head
	for fast != nil && fast.Next != nil {
		slow = slow.Next
		fast = fast.Next.Next
	}
	return slow
}
```

### [19. 删除链表的倒数第 N 个结点](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/)

>先让fast前进n个节点,然后fast和slow共同前进当fast为nil时slow即为要删除节点的上一个节点

![delnhandler](../../../img/delnhandler.gif)

```go
func removeNthFromEnd(head *ListNode, n int) *ListNode {
	vor := &ListNode{Next: head}
	slow, fast := vor, head
	for ; n > 0; n-- {
		fast = fast.Next
	}
	for fast != nil {
		fast = fast.Next
		slow = slow.Next
	}
	if slow.Next != nil {
		slow.Next = slow.Next.Next
	}
	return vor.Next
}
```

### [160. 相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/)

>错的人一定会错过,对的人终究会重逢.

![doubleponinter](../../../img/doubleponinter.png)

```go
func getIntersectionNode(headA, headB *ListNode) *ListNode {
    if headA == nil || headB == nil {
        return nil
    }
    pointerA,pointerB := headA,headB
    for pointerA != pointerB {
        if pointerA == nil {
            pointerA = headB
        } else {
            pointerA = pointerA.Next
        }
        if pointerB == nil {
            pointerB = headA
        } else {
            pointerB = pointerB.Next
        }
    }
    return pointerA
}
```

## 左右指针

>左右指针最经典用法即二分查找

### [35. 搜索插入位置](https://leetcode.cn/problems/search-insert-position/)

```go
func searchInsert(nums []int, target int) int {
	left, right := 0, len(nums)-1
	for left <= right {
		middle := (right-left)>>1 + left
        if nums[middle] == target {
            return middle
        }
		if nums[middle] < target {
			left = middle + 1
		} 
        if nums[middle] > target {
			right = middle - 1
		}
	}
	return left
}
```

### [287. 寻找重复数](https://leetcode.cn/problems/find-the-duplicate-number/)

>这里的left和right不是索引值而是值域
>
>题目规定了数组内的数是从1到长度-1的数且一定有重复数
>
>例如这个数组长度为5,分别为1,3,4,2,2
>
>定义left为1right为4,即midd为2;判断数组小于等于midd的元素有多少,如果小于等于midd说明重复数一定是在left到midd中间的反之亦然,1,3,4,2,2中小于等于2的有三个,3 > 2重复数一定在1到2中

```go
func findDuplicate(nums []int) int {
    left,right := 1,len(nums)-1
    for left <= right {
        count := 0
        midd := (right - left) >> 1 + left
        for _,num := range nums {
            if num <= midd {
                count++
            }
        }
        if count <= midd {
            left = midd + 1
        } else {
            right = midd - 1
        }
    }
    return left
}
```
