POST 创建文档#

POST /shopping/_doc

Body 请求参数

{
  "title": "小米手机",
  "category": "小米",
  "image": "http://abc.com/a.jpg",
  "price": 2999.99
}

请求参数#

名称 位置 类型 必选 说明
body body object 否 none

返回示例

成功

{
  "_index": "shopping",
  "_id": "PAGtQ4kBKLdwdVo2W1at",
  "_version": 1,
  "result": "created",
  "_shards": {
    "total": 2,
    "successful": 1,
    "failed": 0
  },
  "_seq_no": 0,
  "_primary_term": 1
}

» _index string true none none
